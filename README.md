# GitHub Pages Başlangıç
Bu repo, GitHub Pages üzerinde basit bir statik site yayına almak için hazırlanmıştır.

## Hızlı Kullanım
1. Bu klasörü GitHub’da yeni bir depoya yükleyin.
2. **Settings → Pages**: Source = `Deploy from a branch`, Branch = `main` (root).
3. (Opsiyonel) Özel alan adı: `www.tethersecure.com.tr` için CNAME → `<kullanici>.github.io`.
